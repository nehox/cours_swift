# Cours Swift

Ce dépôt regroupe les supports et exercices utilisés pour découvrir Swift et SwiftUI en playground.

## Contenu

| Élément | Objectif |
| --- | --- |
| `playground/base.playground` | Revoir les bases du langage Swift |
| `playground/swiftUI.playground` | Découvrir les premiers composants et mécanismes de SwiftUI |
| `prez_v2 (1).key` | Support de présentation du cours |

## Parcours conseillé

Commencez par `playground/base.playground`, puis poursuivez avec `playground/swiftUI.playground`.

## Sujets abordés

### `base.playground`

Ce playground permet de travailler les fondamentaux de Swift :

1. variables et constantes
2. types de base
3. conditions et boucles
4. fonctions
5. collections (`Array`, `Dictionary`)
6. optionnels
7. structures et instances
8. énumérations

### `swiftUI.playground`

Ce playground introduit les bases de SwiftUI :

1. création d'une vue simple
2. mise en page avec `VStack`
3. affichage d'images système
4. boutons et interactions
5. listes
6. gestion de l'état
7. navigation entre vues
8. formulaires
9. onglets avec `TabView`
10. présentation modale avec une feuille

## Conseils de travail

- avancez exercice par exercice
- testez souvent dans le playground
- essayez de modifier les exemples pour observer le comportement de Swift
- gardez un code simple et lisible avant d'ajouter des variantes
